# 
Code for Reproducing Results in Paper 'Deep Hyperspectral Inpainting with the Sparsity and Low Rank Considerations' 
For LRS-PnP (i,e. Algorithm 1 in the paper), both Matlab and Python IMmplementation are provided.
For LRS-PnP-DIP/LRS-PnP-DIP-1-Lip(i,e. Algorithm 2 in the paper), only Python implementation is provided.
Feel free to play around on the different images and hyperparameters.
